# Task: Module Installation & Configuration Verification

## Description
Verify that all three ECPay modules can be installed/upgraded successfully on the live Odoo 18 instance. Configure ECPay test credentials and verify all menu items, views, and settings pages load correctly.

## Acceptance Criteria
- [ ] All three modules upgrade without errors
- [ ] ECPay invoice settings page accessible and configurable
- [ ] ECPay payment provider settings accessible and configurable
- [ ] All menu items load correctly
- [ ] All views render without errors
- [ ] ECPay test mode enabled
- [ ] Screenshots captured for each configuration step

## Technical Details

### Modules to Install/Upgrade
1. `ecpay_invoice_tw` - Taiwan Electronic Invoice
2. `ecpay_invoice_website` - Website Invoice Frontend
3. `payment_ecpay` - Payment Provider

### Configuration Steps

**ECPay Invoice Configuration:**
1. Navigate to Settings → General Settings → ECPay Invoice
2. Enable ECPay Demo Mode (test environment)
3. Configure MerchantID, HashKey, HashIV (test credentials)
4. Set invoice creation mode (manual/automatic)

**ECPay Payment Configuration:**
1. Navigate to Website → Configuration → Payment Providers
2. Find ECPay provider and enable
3. Configure MerchantID, HashKey, HashIV (test credentials)
4. Enable desired payment methods (credit card, ATM, CVS, etc.)
5. Set domain name for callbacks

### Menu Items to Verify
- Invoicing → ECPay menu items
- Settings → ECPay configuration
- Website → Payment Providers

### Playwright Test Steps
1. Login as admin
2. Navigate to Apps → Search "ecpay"
3. Upgrade each module
4. Verify no error notifications
5. Navigate to each settings page
6. Take screenshots
